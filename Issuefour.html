Usage in Current Case Scenario

The state of calamities and disaster is a tough time. Countless people get strucked in such situations.

As a measure of relief, we can use the Mission Mitigation app to reduce the panic and chaos in the situation.

Here API might play an important role here. In the Mitigation app, we have mentioned featues like Weather alerts, SOS, Tracking, Positioning etc. Here API's can give a good quality inputs here.



Here Tracking API can help in understanding the disaster prone area as well as getting the best routes for the helping hands or the Rescue Team Heading in that directions.

Here Positioning API can be used in positioning of the current pinpoint location of the user. This could help the Rescue team to look at the exact location which could safe lives.

Destination Weather API can help us in proving alerts and warnings about the weather changes at the place and could help the user in judging the current scenario.

Here MAP API can be used with the XYZ API to assist in evacuating the place in the best possible route.

Positioning API

The HERE Positioning API enables developers to build applications that require location estimates based on radio network measurement data. The supported measurement data includes 2G, 3G, 4G, and WLAN measurements. We could even include the GPS data that we could fetch out here to locate the exact user locations.
